# Digital Obsidian Garden
This is the repository for my personal website for student ressources, as well as articles on various chemical and physical subjects so that I may hope to remember them. 

This is a modified version of the template to be used together with the [Digital Garden Obsidian Plugin](https://github.com/oleeskild/Obsidian-Digital-Garden), and is deployed with vercel.
See the README in the plugin repo for information on how to set it up. The userSetup.js has been modified to support markup links between pages that do not appear on the graph view.
